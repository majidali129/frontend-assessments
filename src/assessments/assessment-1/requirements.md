**Frontend Practice Assessment (React + JavaScript)**
Build a simple **Task Manager App** in React.
**Requirements:**

- Display a list of tasks
- Add a new task
- Mark task as completed
- Delete a task
- Search tasks by title
- Show total and completed task count
  **Technical Expectations:**
- Use functional components
- Use React hooks
- Write clean and reusable code
- Keep basic responsive styling
  **Bonus:**
- Use localStorage
- Add filters: All / Completed / Pending
- Handle empty input validation
